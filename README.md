### 🚀 Huo-Tian C++ Utility Library (HuoTCUL)

**一个轻量级、开箱即用的 C++ 常用性扩展库。(仍在继续更新！)**

本库由 **Huo-Tian & Huotian2016** 开发，旨在为 C++ 开发者（特别是算法竞赛选手和嵌入式开发者）提供一个纯头文件、零依赖且性能优异的基础工具集。

### ✨ 核心特性

* **纯头文件（Header-only）**：无需复杂的编译和链接，只需 `#include` 即可使用。
* **高精度运算**：内置强大的 `GJD` 类，支持任意位数的整数加减乘除。
* **通用排序算法**：提供多种模板化排序接口，不仅支持基础数据类型，也能直接对你的自定义类进行排序。
* **零外部依赖**：仅依赖 C++ 标准库，兼容 GCC、Clang 和 MSVC 等主流编译器。

### 🛠️ 快速上手

在你的 C++ 项目中引入本库极其简单：

1. 将仓库中的 `include` 文件夹（或对应的 `.h` 头文件）复制到你的项目目录。
2. 在代码中引入头文件即可。
